## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {lgplus-client, ".*",
      {git, "git@github.com:YOURNAME/lgplus-client.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
